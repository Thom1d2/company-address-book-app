Written on Apple Mac os using python 3.6 and django

run in terminal after activating the local environment

$ source myvenv/bin/activate

followed by 'python manage.py runserver' (in the folder directory as below)

(myvenv) <your computer>: addresstree <user>$ python manage.py runserver

this should run a local server at 'http://127.0.0.1:8000/'

for admin privilages go to 'http://127.0.0.1:8000/admin/'
